# CURD APP
## TECH STACK USED TO BUILT
- ### Backend : Nodejs
- ### Frontend : Angularjs, Bootstrap

## Run Locally
### Clone the Repo
```
git clone git@github.com:Pankaj-SinghR/CURD-App-NodeJs-AngularJs.git
```
## Change directory to backend server
```
cd CURD-App-NodeJs-AngularJs/curd-app/backend
```


## Run Node server

```
node index.js
```
